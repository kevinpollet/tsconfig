# @kevinpollet/tsconfig

Shareable TypeScript project configuration.

## Install

```bash
$ npm install --save-dev @kevinpollet/tsconfig
```

## Extend configuration

```json
{
  "extends": "@kevinpollet/tsconfig"
}
```
